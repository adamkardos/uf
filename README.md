# uf
Simple FORM program to calculate the first and second Symanzik polynomials

## Usage
To see how the package works, please check the examples in the check folder.

## Note
At the moment the sign convention is arbitrary and the code can 
only calculate the U and F polynomials for a single graph at a time.
